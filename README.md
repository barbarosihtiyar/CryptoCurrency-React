# CryptoCurrency
* Cryptocurrency is a React based web application.On this site, users can view the cryptocurrencies they want. You can get information about cryptocurrencies.*

# Features
- User text input
- Instant price
- Over 80 cryptocurrencies
- Crypto market value
- 24-hour change
